# Contributing

Silt is currently an experimental research project. Public feedback is welcome,
but support and maintenance response are not guaranteed.

## Issues

Please use GitHub Issues for:

- bug reports with a minimal reproduction
- documentation corrections
- questions about the current public claim boundary
- small, concrete feature requests

An issue may be closed without implementation if it does not fit the current
research direction, cannot be reproduced, or would widen the public claim beyond
the evidence available in the repository.

## Pull Requests

Silt is not currently accepting outside pull requests as the normal contribution
path. Opening an issue is the preferred route.

If an issue is accepted, the maintainers may implement the change from that
issue in the repository. This does not guarantee that every accepted issue will
be scheduled, implemented, or released.

Maintainer pull requests are used as the repository's default change-management
unit. See [MAINTAINERS.md](MAINTAINERS.md) for the maintainer branch, review,
and verification workflow.

## Security And Production Use

Silt is not ready for production use. Do not rely on it for security-sensitive
or availability-sensitive systems.
