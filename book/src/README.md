# The Silt Programming Language

Siltは、低レイヤーの事実を型付きの値として扱うための言語です。

S式の表面構文、依存型コア、数量付きの値使用、明示的なメモリ表現、freestanding targetへの降下を、一つの言語像として結びつけます。目指しているのは、機械に近いコードを、ただ危険な命令列として書くのではなく、読める形、検査できる形、段階的に強められる形で書くことです。

この本は、Siltを「構文を眺める対象」ではなく「低レイヤー実装を型付きに組み立てる道具」として学ぶための入門書です。読者は、基本的なプログラミング、関数型の考え方、低レイヤーのメモリ、型による検証の入口を知っているものとします。専門的な依存型理論、OSカーネル実装、コンパイラ実装を前提にはしません。

この本を読み終えるころには、次のことができる状態を目標にします。

- Siltプログラムの基本構造を読める
- `claim` / `def` / `Pi` / `fn` / `match` の関係を説明できる
- 数量 `0`、`1`、`omega` が何を制限するかを理解できる
- `Eff pre post A` による状態付き副作用を読める
- `layout`、`static-cell`、`static-value`、`static-bytes` がどのように低レイヤー表現へ落ちるかを理解できる
- Limine/x86_64/freestanding C/QEMUで扱うSiltの実装境界を説明できる
- `examples/limine.silt` のallocator handoffまでの流れを追える

Siltは研究段階の言語です。そのため、この本では実装済みの機能、検証済みの経路、設計中の領域を分けて扱います。低レイヤーのコードを型付きに書けることと、汎用のカーネル記述言語として完成していることは同じではありません。

この区別は、ただの注意書きではありません。Siltを読むときの基本姿勢です。何が型で守られているのか。何が生成物で確認されているのか。何がQEMU上の観測で支えられているのか。この本では、その層を分けながら進みます。
